## 系统整体架构

```
世界坐标系 (World Frame)
    ↓
    ├─ 定义3个角点：左上 (TL), 右上 (TR), 左下 (BL)
    │
    ├─ GridFrame 生成器
    │  ├─ 计算格子宽度/高度
    │  └─ 生成 3×3 九个格子
    │     └─ 每个格子：4角点 + 1中心点 (3D 世界坐标)
    │
    ↓
相机坐标系 (Camera Frame)
    ├─ 刚体变换 (Pose 从里程计)
    │  T_world_to_camera = relative_pose
    │
    ↓
鱼眼相机投影 (Fisheye Camera Model)
    ├─ Kannala-Brandt 畸变模型
    │  ├─ 计算 3D点的入射角 θ = atan2(√(x²+y²), z)
    │  ├─ 应用角度畸变：θ_d = θ×(1 + k₁θ² + k₂θ⁴ + k₃θ⁶ + k₄θ⁸)
    │  └─ 映射到像素平面 (u, v)
    │
    ↓
图像坐标系 (Image Frame)
    ├─ 2D 畸变投影点
    ├─ 4个角点 → 4条贝塞尔曲线段
    └─ 绘制曲线九宫格
```

## 第一部分：3D九宫格生成

### 1.1 输入数据

用户提供**三个关键点**（世界坐标系）：
- **P₁（左上格子中心）**：格子 (0,0) 的中心
- **P₃（右上格子中心）**：格子 (0,2) 的中心
- **P₇（左下格子中心）**：格子 (2,0) 的中心

```yaml
grid:
  corner_top_left: [0.0, 0.0, 0.0]        # 外角点 (自动计算或手动输入)
  corner_top_right: [0.9, 0.0, 0.0]       # 外角点
  corner_bottom_left: [0.0, 0.9, 0.0]     # 外角点
  rows: 3           # 行数
  cols: 3           # 列数
```

### 1.2 格子生成算法

**步骤1：计算方向向量**

```
vec_right = TR - TL  （横向向量）
vec_down = BL - TL   （纵向向量）
```

**步骤2：计算单位方向向量**

```
total_width = ||vec_right||
total_height = ||vec_down||

unit_right = vec_right / total_width
unit_down = vec_down / total_height
```

**步骤3：计算每个格子的尺寸**

```
cell_width = total_width / cols
cell_height = total_height / rows
```

**步骤4：生成每个格子的4个角点**

对于第 (row, col) 个格子：

```
corner_TL = TL + unit_right × (col × cell_width) + unit_down × (row × cell_height)
corner_TR = TL + unit_right × ((col+1) × cell_width) + unit_down × (row × cell_height)
corner_BR = TL + unit_right × ((col+1) × cell_width) + unit_down × ((row+1) × cell_height)
corner_BL = TL + unit_right × (col × cell_width) + unit_down × ((row+1) × cell_height)
```

**数学表达式：**

对于任意点 (i, j)，世界坐标为：
$$P_{i,j} = TL + i \cdot unit_{right} \cdot cell_{width} + j \cdot unit_{down} \cdot cell_{height}$$

### 1.3 示例计算

假设：
- TL = [0, 0, 0]
- TR = [0.9, 0, 0]
- BL = [0, 0.9, 0]
- rows=3, cols=3

则：
- `cell_width = 0.9/3 = 0.3`
- `cell_height = 0.9/3 = 0.3`
- `unit_right = [1, 0, 0]`
- `unit_down = [0, 1, 0]`

**格子(1,1)的4个角点：**
```
corner_TL = [0.3, 0.3, 0]
corner_TR = [0.6, 0.3, 0]
corner_BR = [0.6, 0.6, 0]
corner_BL = [0.3, 0.6, 0]
```

## 第二部分：鱼眼畸变模型

### 2.1 Kannala-Brandt 鱼眼模型（OpenCV）

这是 `cv2.fisheye` 使用的标准模型，也是你的系统实现的模型。

**畸变公式：**

$$\theta_d = \theta + k_1 \theta^3 + k_2 \theta^5 + k_3 \theta^7 + k_4 \theta^9$$

其中：
- $\theta$：3D点相对于光轴的入射角（弧度）
- $\theta_d$：畸变后的角度（用于像素映射）
- $k_1, k_2, k_3, k_4$：畸变系数（通过标定得到）

### 2.2 标定过程

**标定工具链：**

```
calibrate_fisheye_180.py
    ↓
1. 采集20-30张棋盘格标定图像
2. 检测棋盘角点(10×7)
3. 调用 cv2.fisheye.calibrate()
    ├─ 输入：对象点(3D)、图像点(2D)、图像尺寸
    └─ 输出：K矩阵、D向量(k1,k2,k3,k4)
4. 保存到 fisheye_calibration.json
```

**标定输出示例：**

```json
{
  "camera_matrix": {
    "data": [
      [595.3, 0, 960],
      [0, 595.1, 540],
      [0, 0, 1]
    ]
  },
  "distortion_coefficients": {
    "data": [
      -0.1234,    // k1
      -0.0567,    // k2
       0.0012,    // k3
      -0.0008     // k4
    ]
  },
  "image_width": 1920,
  "image_height": 1080
}
```

**参数含义：**

| 参数 | 符号 | 作用 | 典型值（180°镜头）|
|------|------|------|------------------|
| 焦距x | fx | 像素尺度(x方向) | ~595 |
| 焦距y | fy | 像素尺度(y方向) | ~595 |
| 主点x | cx | 主点水平位置 | ~960 |
| 主点y | cy | 主点竖直位置 | ~540 |
| 畸变系数1 | k1 | 主畸变项 | -0.1 ~ -0.3 |
| 畸变系数2 | k2 | 高阶畸变 | ±0.05 |
| 畸变系数3 | k3 | 更高阶畸变 | ±0.01 |
| 畸变系数4 | k4 | 最高阶畸变 | ±0.001 |

## 第三部分：3D到2D的投影流程

### 3.1 完整投影管道

```
3D World Point (P_w)
    ↓ [刚体变换]
3D Camera Point (P_c) = T_c_w × P_w
    ↓ [极坐标转换]
入射角 θ, 方向 (u_x, u_y)
    ↓ [Kannala-Brandt 畸变]
畸变角 θ_d
    ↓ [投影到图像平面]
2D Image Point (u, v)
```

### 3.2 详细步骤

#### 步骤1：刚体变换（世界→相机坐标系）

```python
T_w_c = relative_pose_from_odometry  # 4×4 变换矩阵

# 相机坐标系下的3D点
P_c = T_w_c @ P_w  # 3D homogeneous coordinates
x, y, z = P_c[:3]
```

**数学表达：**
$$P_c = R_{w \to c} P_w + t_{w \to c}$$

其中 $R$ 是旋转矩阵，$t$ 是平移向量。

#### 步骤2：计算入射角

$$r_{3D} = \sqrt{x^2 + y^2}$$
$$\theta = \arctan2(r_{3D}, z)$$

**几何含义**：
- $r_{3D}$：空间点在 XY 平面的投影距离
- $z$：点沿光轴的深度
- $\theta$：从光轴的夹角（0° 正对着镜头，90° 在侧面，180° 在镜头后方）

#### 步骤3：应用 Kannala-Brandt 畸变

$$\theta^2 = \theta \times \theta$$
$$\theta^4 = \theta^2 \times \theta^2$$
$$\theta^6 = \theta^4 \times \theta^2$$
$$\theta^8 = \theta^6 \times \theta^2$$

$$\theta_d = \theta \times \left(1 + k_1 \theta^2 + k_2 \theta^4 + k_3 \theta^6 + k_4 \theta^8\right)$$

**畸变效果示意：**

```
正常鱼眼（k1 < 0):
    未畸变 θ = 45° → 畸变 θ_d = 35°（向中心压缩）
    
极端鱼眼（k1 = -0.3):
    θ = 60° → θ_d ≈ 40°（大幅向中心压缩）
    θ = 80° → θ_d ≈ 55°（更强压缩）
```

#### 步骤4：映射到像素坐标

**情况1：点在光轴上** ($r_{3D} < 1e-10$)

```
u = cx
v = cy
```

**情况2：点在光轴外**

```python
# 归一化方向
u_n = x / r_3d
v_n = y / r_3d

# 投影到像素
u = fx × θ_d × u_n + cx
v = fy × θ_d × v_n + cy
```

**数学表达：**
$$u = f_x \theta_d \frac{x}{r_{3D}} + c_x$$
$$v = f_y \theta_d \frac{y}{r_{3D}} + c_y$$

### 3.3 代码实现

```python
class KannalaBrandtFisheye(CameraModel):
    def project_point(self, point_3d):
        x, y, z = point_3d[0], point_3d[1], point_3d[2]
        
        # 步骤1：计算入射角
        r_3d = math.sqrt(x*x + y*y)
        theta = math.atan2(r_3d, z)
        
        # 检查视场范围
        if theta > math.pi:
            return None
        
        # 步骤2：应用畸变
        theta2 = theta * theta
        theta4 = theta2 * theta2
        theta6 = theta4 * theta2
        theta8 = theta6 * theta2
        
        theta_d = theta * (1 + k1*theta2 + k2*theta4 + k3*theta6 + k4*theta8)
        
        # 步骤3：投影到像素
        if r_3d < 1e-10:
            u = cx
            v = cy
        else:
            u_n = x / r_3d
            v_n = y / r_3d
            u = fx * theta_d * u_n + cx
            v = fy * theta_d * v_n + cy
        
        return (u, v)
```

### 3.4 批量投影

对于九宫格，需要投影多个点：

```python
# 获取所有格子的角点
corners_world = grid_frame.get_all_corners_world()  # [P1, P2, ... P36]

# 批量投影
corners_image = []
for P_w in corners_world:
    # 世界→相机坐标系
    P_c = T_w_c @ P_w
    
    # 投影
    pixel = camera_model.project_point(P_c)
    if pixel:
        corners_image.append(pixel)
    else:
        corners_image.append(None)
```

## 第四部分：曲线插值与绘制

### 4.1 从直线到曲线

由于鱼眼畸变的非线性性质，**直线不再是直线**。

**原因分析：**

```
直线的3个点：P1, P2, P3（共线）

投影后：
  P1' = project(P1) = (u1, v1)
  P2' = project(P2) = (u2, v2)
  P3' = project(P3) = (u3, v3)

由于鱼眼畸变函数的非线性：
  P1', P2', P3' 通常不共线 → 形成曲线
```

### 4.2 贝塞尔曲线拟合

对于每条格子边，使用三次贝塞尔曲线拟合：

$$B(t) = (1-t)^3 P_0 + 3(1-t)^2 t P_1 + 3(1-t) t^2 P_2 + t^3 P_3, \quad t \in [0,1]$$

**4个控制点：**

```
P0 = 端点1投影
P1 = 控制点1（插值估计）
P2 = 控制点2（插值估计）
P3 = 端点2投影
```

### 4.3 简化方法：等间距点插值

不使用解析贝塞尔曲线，而是用多条短直线段近似：

```python
def draw_curved_line(img, P_start, P_end, camera_model, T_w_c, num_segments=20):
    """绘制从P_start到P_end的曲线（沿直线参数化）"""
    
    points = []
    for i in range(num_segments + 1):
        # 在3D直线上等间距取点
        t = i / num_segments
        P_3d = (1-t) * P_start + t * P_end
        
        # 变换到相机坐标系
        P_c = T_w_c @ P_3d
        
        # 投影到2D
        pixel = camera_model.project_point(P_c)
        if pixel:
            points.append(pixel)
    
    # 用直线段连接所有点
    for i in range(len(points) - 1):
        cv2.line(img, tuple(map(int, points[i])), 
                      tuple(map(int, points[i+1])), 
                      color=(255, 0, 0), thickness=2)
```

### 4.4 绘制九宫格

```python
def draw_grid(img, grid_frame, camera_model, T_w_c):
    """绘制九宫格曲线"""
    
    # 遍历所有格子
    for cell in grid_frame.cells.values():
        corners_world = cell.corners_world  # [TL, TR, BR, BL]
        
        # 绘制4条边
        edge_pairs = [(0,1), (1,2), (2,3), (3,0)]
        
        for start_idx, end_idx in edge_pairs:
            P_start = corners_world[start_idx]
            P_end = corners_world[end_idx]
            
            draw_curved_line(img, P_start, P_end, 
                           camera_model, T_w_c, num_segments=15)
        
        # 绘制中心点
        center_c = T_w_c @ cell.center_world
        pixel = camera_model.project_point(center_c)
        if pixel:
            cv2.circle(img, tuple(map(int, pixel)), 
                      radius=4, color=(0, 0, 255), thickness=-1)
```

## 完整数学推导

### 推导1：从中心点到外角点的转换

**输入：** 三个格子中心点 P₁(左上), P₃(右上), P₇(左下)

**输出：** 三个外角点 TL, TR, BL

**推导过程：**

对于 $3 \times 3$ 九宫格：
- P₁ 到 P₃ 跨越 $(3-1)=2$ 个格子宽度
- P₁ 到 P₇ 跨越 $(3-1)=2$ 个格子高度

$$\vec{v}_{right} = P_3 - P_1 \quad \text{（表示2个格子宽度的向量）}$$
$$\vec{v}_{down} = P_7 - P_1 \quad \text{（表示2个格子高度的向量）}$$

$$\vec{cell}_{width} = \frac{\vec{v}_{right}}{2}$$
$$\vec{cell}_{height} = \frac{\vec{v}_{down}}{2}$$

外角坐标：
$$TL = P_1 - \frac{\vec{cell}_{width}}{2} - \frac{\vec{cell}_{height}}{2}$$
$$TR = P_3 + \frac{\vec{cell}_{width}}{2} - \frac{\vec{cell}_{height}}{2}$$
$$BL = P_7 - \frac{\vec{cell}_{width}}{2} + \frac{\vec{cell}_{height}}{2}$$

### 推导2：鱼眼畸变的物理意义

Kannala-Brandt 模型本质上是"**角度空间**"的多项式畸变：

**标准投影（无畸变）：**
$$r = f \theta$$

**有畸变投影：**
$$r_d = f \theta_d = f[\theta + k_1\theta^3 + k_2\theta^5 + ...]$$

**对于 180° 鱼眼镜头**（典型 $k_1 \approx -0.2$）：

- $\theta = 0°$ (正对着) → $\theta_d = 0°$
- $\theta = 30°$ → $\theta_d \approx 28°$ (轻微内缩)
- $\theta = 60°$ → $\theta_d \approx 52°$ (明显内缩)
- $\theta = 80°$ → $\theta_d \approx 62°$ (强烈内缩，但仍在视场内)

**结果：从中心向边缘，投影点密度增加（内缩效应）**

### 推导3：曲线形成的原因

**定理：** 在非线性投影下，参数直线变为参数曲线

**证明：**

设 $P(t) = (1-t)P_A + tP_B$，$t \in [0,1]$ 是3D直线

在相机坐标系下：
$$P_c(t) = T_{w \to c} [P(t)]$$

投影后：
$$P_{img}(t) = \pi(P_c(t))$$

其中 $\pi$ 是投影函数（包含 Kannala-Brandt 畸变）。

若 $\pi$ 非线性，则 $P_{img}(t)$ 是关于 $t$ 的曲线（通常为三次曲线）。

**特殊情况：** 过光轴的直线投影为直线

$$\text{if } P(t) \text{ passes through optical axis, then } P_{img}(t) \text{ is linear}$$

## 代码流程示例

### 完整处理管道

```python
import cv2
import numpy as np
from ar_grid_detector_py.camera_models import KannalaBrandtFisheye, CameraIntrinsics
from ar_grid_detector_py.grid_generator import GridFrameGenerator

# ================== 初始化 ==================

# 1. 加载鱼眼相机模型
intrinsics = CameraIntrinsics(
    fx=595.3, fy=595.1,
    cx=960, cy=540,
    width=1920, height=1080,
    k1=-0.1234, k2=-0.0567, k3=0.0012, k4=-0.0008
)
camera = KannalaBrandtFisheye(intrinsics)

# 2. 生成九宫格（3×3）
grid = GridFrameGenerator.generate_nine_grid(
    corner_top_left=[0.0, 0.0, 0.0],
    corner_top_right=[0.9, 0.0, 0.0],
    corner_bottom_left=[0.0, 0.9, 0.0]
)

# 3. 读取里程计（相机位姿）
# 假设：机器人离九宫格 1.5 米，俯视角 30°
T_w_c = np.array([
    [1, 0, 0, 0],
    [0, np.cos(np.deg2rad(30)), np.sin(np.deg2rad(30)), 1.5],
    [0, -np.sin(np.deg2rad(30)), np.cos(np.deg2rad(30)), 0],
    [0, 0, 0, 1]
], dtype=np.float64)

# ================== 投影处理 ==================

def project_grid_and_draw(image, grid, camera, T_w_c):
    """投影九宫格并绘制"""
    
    for cell in grid.cells.values():
        corners_world = cell.corners_world
        
        # 绘制每条边的曲线
        for i in range(4):
            p_start = corners_world[i]
            p_end = corners_world[(i+1) % 4]
            
            # 沿3D直线等间距取点，然后投影
            segment_points = []
            for t in np.linspace(0, 1, 20):
                p_3d = (1-t) * p_start + t * p_end
                p_c = T_w_c @ np.append(p_3d, 1)
                pixel = camera.project_point(p_c[:3])
                
                if pixel and camera.is_in_image(pixel[0], pixel[1]):
                    segment_points.append((int(pixel[0]), int(pixel[1])))
            
            # 用直线段连接
            for j in range(len(segment_points) - 1):
                cv2.line(image, segment_points[j], segment_points[j+1],
                        color=(255, 0, 0), thickness=2)
        
        # 绘制格子中心
        center_c = T_w_c @ np.append(cell.center_world, 1)
        pixel = camera.project_point(center_c[:3])
        if pixel and camera.is_in_image(pixel[0], pixel[1]):
            cv2.circle(image, (int(pixel[0]), int(pixel[1])),
                      radius=5, color=(0, 0, 255), thickness=-1)

# ================== 执行 ==================

# 读取实际图像
img = cv2.imread("fisheye_image.jpg")

# 投影并绘制
project_grid_and_draw(img, grid, camera, T_w_c)

# 显示
cv2.imshow("AR Grid Overlay", img)
cv2.waitKey(0)
```

## 常见问题排查

### Q1：投影点在图像边界外

**原因：** 3D点超出相机视场范围

**解决：**
1. 检查 `theta > π` 判断（> 180° 无法投影）
2. 减小九宫格距离或调整相机位置

### Q2：曲线看起来是直线

**原因：** 畸变系数太小或点离光轴太近

**检查：**
```python
# 打印实际的畸变效果
theta_values = [10, 30, 60, 80]  # 度
for theta_deg in theta_values:
    theta = np.deg2rad(theta_deg)
    theta_d = distort_theta(theta)
    print(f"θ={theta_deg}° → θ_d={np.rad2deg(theta_d):.1f}°")
```

### Q3：投影不准确

**排查步骤：**
1. 验证相机标定参数（用棋盘板测试）
2. 验证外参 `extrinsic_rpy_xyz`
3. 验证九宫格的3个角点坐标
4. 打印中间变量进行单步调试

## 总结

您的鱼眼模型实现包含以下关键流程：

1. **3D格子生成**：从3个角点生成36个3D点
2. **坐标变换**：根据里程计将点转到相机坐标系
3. **鱼眼投影**：使用 Kannala-Brandt 模型进行非线性投影

这个流程充分利用了鱼眼相机的特性，能够准确表示极端视角下的定位标记。
